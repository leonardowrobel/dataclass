# dataclass

Instructions:
- First you need:
	- PHP 5.4 or greater with the command-line interface (CLI) and JSON extension installed
	- The Composer dependency management tool
	- A Google account
- Then execute "composer require google/apiclient:^2.0" on project's folder
- To run the application, you need a valid credentials.json file from console.developers.google (see link for further details), put this file on the root project
- You also need to share the spreadsheet with your service account email from credentials.json
- Run "php dataclass.php"

ref:
- Google Sheets and PHP: https://youtu.be/iTZyuszEkxI
- https://developers.google.com/sheets/api/quickstart/php
